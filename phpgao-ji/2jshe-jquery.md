## JS变量的定义

- 如果重新声明Javascript变量，该变量的值不会丢失。

- JS变量均为对象。当您声明一个变量时，就创建了一个新的对象。

```
var a = 1;

var a;
```


## 内置对象

```
var pi = 3.14;
var num = new Number(3.14);
var num = Number(3.14);
```
